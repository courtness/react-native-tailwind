## ⚡️ Quick start

1. **Install**

   Clone and install the dependencies as normal:

   ```
   git clone https://github.com/loveandmoney/flip.git
   yarn install
   ```

2. **Serve**

   Default start command will leave Metro running:

   ```
   yarn start
   ```

3. **Simulate**

   In a separate terminal you can run simulators:

   ```
   yarn run:ios
   ```
